# Technical requirements
- Simple HTML, CSS and JavaScript
- No additional UI frameworks or CSS preprocessors
- Folder structure follows the page hierarchy

# Structure and Navigation
- one single main page
- multiple sub-pages
- user can navigate from the main to a sub-page
- user can navigate from a sub-page back to the main

# Content
- Personal home page used for job search

# Style
- light, optimistic
- Not purely technical but with technical undertones
- Visually interesting
- Easy and fast to comprehend

# Audience
- Recruiters with no or very limited technical knowledge
- Team leaders or department leader with average to good technical knowledge
